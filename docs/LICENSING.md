# Lisensi & Pengecualian — Raporkumer

Proyek ini menggunakan lisensi khas (custom license) yang mengizinkan penggunaan, modifikasi, dan
distribusi untuk keperluan nonkomersial. Beberapa aset statis memiliki lisensi terpisah.

## File lisensi utama

- File `LICENSE` di root repositori berisi teks lisensi lengkap.

## Cakupan Lisensi

Lisensi ini mencakup:

- Seluruh kode sumber dalam repositori
- `static/logo.png`
- `static/logo.ico`
- `static/favicon.ico`
- `static/favicon.png`
- `static/bipmap2.png`

## Pengecualian Aset Statis

Aset di folder `static/` yang tidak tercantum di atas TIDAK dicakup oleh lisensi ini. Aset-aset
tersebut tetap berada di bawah lisensi atau hak tempat penyusun aslinya. Untuk informasi lebih
lanjut, periksa lisensi masing-masing aset atau hubungi tim pengembang.

## Ikon

Ikon di `src/lib/icons` adalah dari proyek Feather (Feather Icons) yang dirilis di bawah lisensi MIT.
Lihat `docs/ICON-CREDITS.md` untuk detail dan tautan sumber.

## Pertanyaan hukum

Dokumen ini hanya ringkasan informasi lisensi untuk kenyamanan pengguna dan pengembang. Untuk
kebutuhan legal/resmi, baca teks lisensi di file `LICENSE` atau konsultasikan penasihat hukum.
